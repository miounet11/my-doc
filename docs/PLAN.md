# mydoc.inping.com Plan

## Purpose
Mirror important OpenClaw-created documents, memory snapshots, and operational docs into a dedicated doc site and Git repo.

## Principles
- local source of truth remains in workspace
- server copy is a mirrored backup / readable site
- GitHub repo `miounet11/my-doc` gets synced copy
- date visibility should come from generated index metadata
- secrets should not be dumped blindly into public pages later

## Initial scope
- workspace bootstrap docs: `SOUL.md`, `USER.md`, `TOOLS.md`, `IDENTITY.md`, `MEMORY.md`
- daily notes under `memory/`
- selected docs under `docs/`
- project docs for active projects, starting with `projects/new2005-finance/docs/`

## Next phase
- generated index page
- publish timestamps
- category folders
- git sync automation
- VPS sync automation
