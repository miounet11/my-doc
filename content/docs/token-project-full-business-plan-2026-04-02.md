## 一、项目总览

本项目围绕 Token 构建一个完整的商业闭环，不只是完成一次销售，而是形成：

- 项目定位与叙事
- Token 机制与销售体系
- 社区增长与运营体系
- 产品承接与应用场景
- 渠道合作与商务拓展
- 中长期变现与生态扩张

Token 的目标不是单点售卖，而是作为项目生态的：

- 用户连接工具
- 激励工具
- 权益承接工具
- 增长工具
- 商业变现工具
- 资本化工具

- 形成持续性商业收入

### 4.2 分配建议框架
建议分为：

- 团队与核心贡献者
- 生态基金
- 社区激励池
- 市场推广池
- 流动性池
- 顾问与战略合作
- 种子轮 / 私募轮
- 公售 / 白名单销售
- 储备金库

### 4.3 锁仓与释放建议
- 团队：长期锁仓 + 线性释放
- 私募：设置 cliff + 分期释放
- 社区激励：按运营节奏释放
- 流动性池：锁仓管理
- 顾问/合作方：避免短期抛压

### 4.4 销售机制建议
可采用：

- 白名单轮
- 种子轮
- 私募轮
- 公售轮
- 渠道联售
- OTC 大客户交易

价格建议采用分层机制：
- 核心支持价
- 渠道合作价
- 市场公开价

## 五、市场规划

### 5.1 用户分层

#### 1）核心种子用户
- 提供早期信任与传播
- 帮助项目完成初始共识构建

#### 2）社区活跃用户
- 提供热度、互动、裂变、传播
- 适合任务激励与活动承接

#### 3）渠道合作方
- 负责用户获取与转化
- 可按分佣或联合销售模式合作

#### 4）KOL / 媒体伙伴
- 放大市场声量
- 增强品牌信任

#### 5）战略合作伙伴
- 带来资金、品牌、用户或场景资源
- 提供长期协同
