# Token 销售说明文档（初稿）

## 1. 项目定位
我们这边正在推进 Token 销售业务，核心目标是围绕项目生态、用户增长、社区建设与长期价值共识，完成 Token 的市场化流通与合作拓展。

本文件用于统一对外沟通口径，方便在与潜在合作方、渠道方、社区用户及投资参与方交流时，清晰说明我们目前的 Token 销售模式、合作方式和基本原则。

---

## 2. 我们在卖什么
我们销售的是项目生态对应的 Token。

Token 的价值不只体现在交易层面，更体现在以下几个方向：
- 生态参与凭证
- 社区共识载体
- 平台发展激励工具
- 后续合作与权益承接基础

也就是说，Token 不是单纯的“卖币”动作，而是围绕项目生态发展进行的资源配置与用户连接。

---

## 3. 销售目标
当前 Token 销售的主要目标包括：

1. **扩大社区基础**  
   吸引首批核心支持者、传播者与合作伙伴进入生态。

2. **增强市场流动性**  
   为后续交易、流通和生态协同打基础。

3. **支持项目发展**  
   通过 Token 销售获取必要资源，用于产品建设、技术开发、市场推广与生态合作。

4. **建立长期共识**  
   让持有者与项目发展方向形成更强绑定，推动长期价值增长。

---

## 4. 面向对象
我们的 Token 销售主要面向以下几类对象：
- 对项目方向认可的社区用户
- 愿意长期参与生态建设的早期支持者
- 有渠道资源或市场资源的合作伙伴
- 关注 Web3、数字资产、项目早期机会的投资参与者

---

## 5. 合作方式
当前可采用的合作方式包括但不限于：

### 5.1 直接购买
合作方或用户可根据项目安排，直接参与 Token 购买。

### 5.2 渠道合作
具备社群、流量、KOL、媒体、社区资源的合作伙伴，可通过渠道分发、联合推广等方式参与销售。

### 5.3 战略合作
对于具备资金、品牌、资源协同能力的合作方，可进一步沟通战略合作、联合生态建设、长期持仓支持等合作模式。

---

## 6. 我们的核心优势
### 6.1 清晰的项目方向
我们不是单纯做短期销售，而是围绕长期项目建设来推进 Token 流通和市场拓展。

### 6.2 生态发展空间
Token 后续将承接更丰富的生态场景、社区权益和合作可能，具备持续扩展能力。

### 6.3 灵活的合作机制
针对不同类型的合作伙伴，我们可以提供更灵活的合作方案，包括销售支持、联合推广、资源置换、阶段性合作安排等。

### 6.4 重视长期关系
我们更看重长期共识，而不是一次性成交。对于真正认可项目方向的合作伙伴，我们希望建立长期稳定的合作关系。

---

## 7. 基本沟通原则
在 Token 销售及合作推进过程中，我们坚持以下原则：

- **信息透明**：关键信息沟通清晰，避免模糊表述。
- **合作共赢**：追求长期价值，不做一次性关系。
- **节奏可控**：根据市场情况、合作对象与项目进展，动态安排销售节奏。
- **长期导向**：以项目长期发展与生态健康为优先。

---

## 8. 适合对外的简版介绍
我们目前正在推进项目 Token 销售，主要面向认可项目方向、愿意参与生态建设和长期合作的用户与伙伴。我们希望通过 Token 销售引入首批核心支持者，同时为后续产品发展、市场拓展和生态建设提供支撑。针对不同类型的合作方，我们也可以提供更灵活的合作模式，包括直接购买、渠道合作和战略合作。
